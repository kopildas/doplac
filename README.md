### Common setup ⚒

Clone the repo and install the dependencies.

<br/>

```bash
git clone https://github.com/kopildas/doplac.git
```

```bash
cd doplac
```

```bash
cd client
```

```bash
npm install
```

```bash
npm run dev
```

###### This will start `frontend` in `http://localhost:3000/`
